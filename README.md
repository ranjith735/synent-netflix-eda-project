# Netflix EDA Project

## Objective
Analyze Netflix dataset using Exploratory Data Analysis.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Data Visualization
4. Insights

## Key Insights
- Movies dominate Netflix
- USA has highest content
- Drama is top genre

## Dataset
