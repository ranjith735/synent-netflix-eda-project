# Netflix EDA Project

## Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix dataset using Python.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* VS Code

## Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Data Visualization
4. Insights Extraction

## Key Insights

* Netflix contains more Movies than TV Shows
* USA has the highest number of Netflix titles
* Drama is one of the most common genres
* Content additions increased rapidly after 2015
